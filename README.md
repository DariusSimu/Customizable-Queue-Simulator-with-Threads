Customizable Queue Simulator with Threads
A Java application that simulates a customizable queuing system using multithreading. The simulation is configured through a setup window where the user can define the following parameters:

Number of queues and clients
Intervals for client arrival time and serving time
Total simulation time
Queue strategy: Time-based or Size-based

Once the simulation starts, clients are dynamically assigned to queues based on the selected strategy, with each queue running on its own thread. The entire simulation log is automatically saved to a .txt file for later review and analysis.
